                   AI-Driven Public Health Chatbot for Disease Awareness


Project Description

The AI-Driven Public Health Chatbot is an intelligent healthcare assistance system that provides disease awareness, symptom information, vaccination details, and public health alerts. Using Artificial Intelligence and Natural Language Processing (NLP), the chatbot understands user queries and delivers relevant healthcare information instantly.

Problem Statement

Many people do not have immediate access to reliable healthcare information. Lack of awareness about diseases, symptoms, preventive measures, and vaccination schedules can lead to delayed diagnosis and health risks. This project aims to provide quick and accurate healthcare information through an AI-powered chatbot.
